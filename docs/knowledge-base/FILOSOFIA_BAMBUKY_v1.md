# FILOSOFÍA BAMBUKY (BORRADOR)

No creemos que una sesión newborn se trate únicamente de fotografías.

Las fotografías son importantes.

Pero lo que realmente recordarán las familias será cómo se sintieron ese día.

Durante las primeras semanas de vida de un bebé todo ocurre muy rápido.

Hay cansancio.

Hay dudas.

Hay miedo.

Hay felicidad.

Hay incertidumbre.

Y de pronto una sesión de fotos se convierte en una pausa.

Un momento para respirar.

Para hablar.

Para compartir experiencias.

Para escuchar.

Para sentirse acompañados.

Durante años hemos descubierto que cada familia necesita algo diferente.

Algunos necesitan hablar de bebés.

Otros necesitan hablar de lactancia.

Otros necesitan hablar de fútbol.

Otros necesitan escuchar que lo están haciendo bien.

Nosotros no intentamos dirigir la conversación.

Intentamos encontrar aquello que esa familia necesita en ese momento.

Creemos que las mejores fotografías nacen cuando las personas se sienten cómodas.

Cuando bajan la guardia.

Cuando dejan de preocuparse por la hora.

Cuando dejan de pensar en la cámara.

Cuando pueden simplemente estar presentes.

Si dentro de 30 años una familia vuelve a ver sus fotografías, esperamos que recuerde algo más que una imagen bonita.

Esperamos que recuerde cómo se sintió ese día.

Tranquila.

Acompañada.

Escuchada.

Y segura de que estaba haciendo lo mejor que podía.
